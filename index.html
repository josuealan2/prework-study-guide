HTML

  • HTML elements are represented by tags and define the structure and content of a webpage.
  • Elements can have attributes in the form of key-value pairs within opening tags to modify their behavior or provide additional information.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS is used to style and format the visual presentation of HTML elements.
  • CSS selectors target HTML elements based on their tag names, classes, IDs, attributes, or relationships with other elements. They specify which elements the CSS rules should apply to.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript supports variables to store data temporarily.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
